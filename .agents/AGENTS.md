# Workspace Rules & Skills Activation

## Fable Method Workflow (`fable-method`)
This workspace incorporates the **Fable Method** (`fable-method`, `fable-loop`, `fable-judge`) skills.
Whenever tackling complex or multi-step coding tasks in this project (`mmmeisa-web`):
1. **Think (`fable-method`)**: Classify the ask, define done with a verifiable plan, and check primary sources/evidence before changing code.
2. **Act (`fable-loop`)**: Make surgical, minimal edits without unintended side effects.
3. **Prove (`fable-judge`)**: Verify every change by execution (`npm run build` or local test) before reporting completion.
